# Bangalore Home Price Prediction Web Application

An end-to-end Machine Learning web application that predicts house prices in Bangalore based on user inputs such as location, square footage, number of bedrooms (BHK), and number of bathrooms.
This project demonstrates the complete ML lifecycle including data preprocessing, model training, backend development using Flask, and frontend integration.
