⚽ Golazo! — Football Visualization Dashboard

A full-stack football analytics project that visualizes top international goal scorers by nation using interactive charts.

🚀 Features
📊 Dynamic bar charts of top 3 goal scorers per country
🌍 Dropdown to select nation
⚡ FastAPI backend serving structured football data
🎨 Clean dark-themed frontend using HTML, CSS, and Chart.js
🔄 Real-time data fetching using JavaScript
