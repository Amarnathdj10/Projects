# 🍷 Wine Quality Prediction App

An end-to-end Machine Learning web application that predicts wine quality based on physicochemical properties.

## 🚀 Live Demo
https://amarnathdj10-projects-wine-quality-predictionapp-eehztw.streamlit.app/
---

## 📌 Overview

This project uses Machine Learning models to classify wine quality into:
- ❌ Bad
- ⚖️ Average
- ✅ Good

Users can input wine characteristics and instantly receive:
- Predicted quality
- Confidence scores
- Explanation of prediction

---

## 🧠 Features

- 🔍 Predict wine quality using ML models 
- 📊 Confidence breakdown (probability visualization)
- 🧠 Explainable AI (rule-based reasoning)
- 🎨 Clean and modern UI using Streamlit
- 🌐 Fully deployed web app

---

## ⚙️ Tech Stack

- **Frontend**: Streamlit
- **Backend**: Python
- **ML Models**:
  - Logistic Regression
  - Random Forest
  - XGBoost (chosen)
- **Libraries**:
  - scikit-learn
  - pandas
  - numpy
  - joblib

---

## 📊 Dataset

- Red & White Wine Quality Dataset
- Features include:
  - Acidity
  - Alcohol
  - pH
  - Sulphates
  - Density
  - etc.
---

## 🧪 Model Workflow

1. Data preprocessing & merging datasets
2. Feature engineering
3. Label classification (Bad / Average / Good)
4. Model training & evaluation
5. Best model selection
6. Deployment using Streamlit

---

## 📈 Model Performance

- Random Forest and XGBoost performed best
- Logistic Regression used as baseline

---

## 💡 Key Insights

- Alcohol level has strong impact on quality
- Volatile acidity negatively affects quality
- Dataset imbalance affects predictions
